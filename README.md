AI Blog Generator
An AI-powered blog generator leveraging LangChain framework with Groq's ChatGroq LLM. It generates structured blog posts—including introduction, detailed content, and conclusion—on any topic. It integrates Wikipedia and optional Google Search tools for enriching content with factual and real-time information.

Features
Fast, asynchronous blog generation with separate sections.

Integration with Wikipedia for factual content and Google Search for current events.

Easy-to-use command-line interface
